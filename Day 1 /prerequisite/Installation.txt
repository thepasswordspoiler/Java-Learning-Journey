Day 1: Prerequisites
📌 Installation Instructions
To start coding in Java, ensure you have the following installed on your system.

1️⃣ Install Visual Studio Code
🔹 Step 1: Visit the VS Code Official Website  https://code.visualstudio.com/
🔹 Step 2: Choose your operating system (Windows/Mac/Linux) and download Visual Studio Code

2️⃣ Install Java (JDK - Java Development Kit)  
🔹 Step 1: Visit the Oracle Java Downloads     https://www.oracle.com/java/technologies/downloads/
🔹 Step 2: Choose your operating system (Windows/Mac/Linux) and download Java JDK
